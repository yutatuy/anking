# Name

暗記王 (Anking)

# DEMO

![anking](https://user-images.githubusercontent.com/46744500/152680426-191010e3-a614-4948-8147-96acc5b809a2.gif)

# Features

- ID, パスワードで会員登録ができる
- ID, パスワードでログインできる
- 単語帳を管理できる
  - 追加できる
  - 編集できる
  - 削除できる
  - プレイできる
  - 単語を追加できる

# Requirement

- docker
- node v17.0.2

# Usage

```
docker-compose up -d
cd frontend/next
yarn dev
```
